# Declare a formspec image button

## Sample

	minetest.register_node(":default:test_imagebutton", {
		description = "test imagebutton",
		tiles = {"aaa.png"},
		groups = {snappy = 3},

		on_rightclick = function(_,_, clicker)
			local formspec = [[
				size[6,6]
				image_button[4,1.75;1,1;heart.png;btn;hi;;;;#A00000FF]
			]]

			local player_name = clicker:get_player_name()
			minetest.show_formspec(player_name, "test_imagebutton", formspec)
		end,
	})

## Arguments

The argument separator is the semicolon.

The value separator in an argument is the comma.

Up to 9 arguments can be provided.

1. X position (float), Y position (float)
2. X size (float), Y size (float)
3. image name
4. name
5. label
6. clipping (true/false)
7. border (true/false)
8. pressed image name
9. color (#RRGGBBAA or color name)

Only the first 5 arguments are mandatory.
